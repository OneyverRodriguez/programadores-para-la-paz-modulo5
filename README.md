# Programadores para la Paz - Módulo 5

Este repositorio reúne los entregables del módulo 5 del programa Programadores para la Paz, centrados en el diseño de una propuesta de solución para el monitoreo comunitario y la atención de afectaciones socioambientales.

## Propósito del proyecto

El objetivo es documentar y estructurar una idea de sistema de veeduría comunitaria que permita:

- registrar reportes de afectaciones desde el territorio;
- visibilizar problemáticas colectivas con evidencia básica;
- automatizar alertas para el equipo de coordinación;
- priorizar funcionalidades de forma progresiva mediante un backlog y una iteración inicial;
- definir una base técnica mínima para la gestión, autenticación y seguimiento de los reportes.

## Avance del proyecto

La propuesta ha evolucionado desde una idea inicial de monitoreo territorial hacia un concepto más estructurado de producto, incorporando los siguientes pilares:

1. Captura de reportes desde la comunidad.
2. Generación automática de alertas para facilitar la respuesta.
3. Definición de prioridades y criterios para avanzar en el desarrollo.
4. Diseño de una interfaz básica para la participación ciudadana.
5. Definición de una estructura inicial de datos y principios de seguridad para proteger la información.

Este repositorio funciona como un registro del proceso de análisis, planeación, diseño y refinamiento de la propuesta durante las semanas 10 y 11.
